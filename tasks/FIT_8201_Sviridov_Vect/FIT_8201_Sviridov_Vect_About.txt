Тулбар:
	New - очистить панели
	Load - загрузить изображение из bmp файла
	Save as... - сохранить изображение из зоны C в bmp файл

	Select - выбор части рисунки из зоны A. если изображение слишком
маленькое, то оно сразу перемещается в зону B
	From C to B - перенести изображение из C в B

	Grayscale - черно-белое
	Negative - инвертировать цвета

	Floyd-Steinberg dithering - дизеринг по алгоритму Флойда-Стейберга
	Ordered dithering - упорядоченный(?) дизеринг

	Double scale - увеличить изображение в 2 раза

	Roberts operator - определение границ с помощью оператора Робертса
	Sobel operator - определение границ с помощью оператора Собеля

	Blur - размытие
	Sharpen - увеличение резкости

	Emboss - тиснение
	Aquarelle - эффект акварели

	Gamma - гамма-коррекция
	Brightness - яркость
	Contrast - контраст

	About - запустить текстовый редактор с этим документом

	Exit - завершить приложение
