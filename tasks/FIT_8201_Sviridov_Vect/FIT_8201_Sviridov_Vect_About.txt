Тулбар:
	New - очистить
	Load - загрузить документ
	Save as... - сохранить сохранить документ

	Grid - включить/выключить сетку
	Chess order - отображение векторов в шахматном порядке

	Field mode - переключение цветное поле/черно-белое
	
	Arrow mode - переключение типа стрелочек

	Zoom state previous - предыдущее положение масштабирования
	Zoom state next - следующее положение масштабирования
	
	Settings - настройки
	Palette - настройки палитры цветов

	About - запустить текстовый редактор с этим документом

	Exit - завершить приложение

Есть отображение вектора под курсором мыши,
выбор область для приближения с помощью прямоугольника.
Перемещаться по состояниям zoom'а можно с помощью кнопок
на тулбаре/меню.

Диалог настроек немодальный, изменения в модели и в диалоге друг с
другом синхронизованы. 
Диалог выбора цветов модальный. Если галочка стоит - то цвет будет
использоваться. Без первоначального выбора цвета галочку поставить нельзя.
По условию необходимо выбрать минимум 5 цветов, это тоже реализовано.

Используется двойная буферизация и сглаживание.

Можно задать до 20 цветов включительно. При изменении палитры
легенда пересчитывается (как и при изменении сетки). Таким образом, если 
хочется уточнить вид текущей области, то достаточно вызвать диалог 
изменения палитры и нажать OK (возможно, добавив или убрав цвета).

Функцию можно изменить в 
FIT_8201_Sviridov_Vect.vect.VectModel.java
где они заданы явно функциями 
fx(double x, doubly y) и fy(double x, doubly y).

Технически можно выбрать область шириной или высотой ноль.
Тогда будет отображена и область шириной в 0 пикселей.
Вернуться в предыдущее состояние всегда можно (через настройки
или историю состояний), поэтому оставил так.

При сложной модели, resize или иное преобразование может занять пару
секунд.
