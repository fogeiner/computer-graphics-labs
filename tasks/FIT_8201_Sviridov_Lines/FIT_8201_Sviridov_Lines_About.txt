Lines, версия 1.1
Валентин Свиридов, ФИТ, группа 8201.
Программа рисования ломаных кривых с использованием библиотеки Java Swing.

1. Есть ли меню?
Да.

2. Что какая кнопка тулбара делает - слева направо:
	2.1 New - создать новый документ
	2.2 Load - загрузить документ из файла
	2.3 Save as... - сохранить файл под определенным именем
	2.4 Exit - завершить приложение
	2.5 Preferences - показать диалог с настройками
	2.6 About - запустить текстовый редактор с открытым файлом 
FIT_8201_Sviridov_About.txt

3. Как строится ПЛ?
	3.1 Очередная точка ставится по клику, т.е. режим резиновой линии при 
отжатых кнопках.
	3.2 Завершение линии по клику правой кнопки.

4. Могут ли быть разные радиусы у разных ПЛ?
Нет.

5. Разрешается ли ресайз окна?
Да.
