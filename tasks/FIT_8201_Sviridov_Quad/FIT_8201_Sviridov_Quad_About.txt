Валентин Свиридов, ФИТ, группа 8201.

Тулбар:
	Load - загрузить документ
	Save as... - сохранить сохранить документ

	Init -- перенести сцену в начальное положение
	Orts -- показать/скрыть орты
	Box -- показать/скрыть бокс
	Objects -- показать/скрыть объекты
	Render -- сделать трассировку, показать результат / 
		перейти к каркасной модели
	Settings - настройки

	About - запустить текстовый редактор с этим документом

	Exit - завершить приложение

Для удобства автоматически загружена одна из моделей.
Источники подвергаются трансформациям как и другие объекты.
Поддерживается проивольное число треугольников, сфер, источников света.
